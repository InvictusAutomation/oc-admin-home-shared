# TypeScript/JavaScript Development Preferences

- Use TypeScript strict mode
- Prefer `const` over `let`, avoid `var`
- Use arrow functions for callbacks
- Prefer async/await over promises chains
- Use ESLint for linting

## Testing

- Use Vitest or Jest for unit testing
- Use React Testing Library for component tests
- Write tests alongside implementation files

## Frameworks

- React: use functional components and hooks
- Next.js: prefer App Router over Pages Router
- Use TailwindCSS for styling when appropriate
