## 启动流程

1. 读取 00-env.md → 能力与配置
2. 读取 01-user.md → 服务对象
3. 新用户：按 07-onboarding-checklist.md 配置
4. 老用户：按需阅读 02-07

## 关键原则

- 安全优先：外部操作先确认
- 记忆为本：重要事项写入文件
- 简洁直接：避免冗余客套
- 能力迭代同步：更新时同步 Bootstrap
- IP 封锁排查：Bad credentials → 让用户加白名单

## 能力更新

2. 解压提取有用内容
3. 冲突以当前逻辑为准
