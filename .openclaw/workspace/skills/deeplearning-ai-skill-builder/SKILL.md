# DeepLearning.AI Skill Builder

## 描述
集成 Andrew Ng 的 DeepLearning.AI Skill Builder，帮助用户评估AI技能、获取个性化学习建议。

## 触发词
- "AI技能评估"
- "深度学习技能"
- "AI学习建议"
- "skill builder"

## 功能
1. 提供 Skill Builder 访问链接
2. 引导用户进行技能评估
3. 解释评估结果和学习路径

## 使用方法
告诉用户访问 https://skillbuilder.deeplearning.ai 进行AI技能评估。

Andrew Ng 创建的这个免费工具可以：
- 评估你的AI技能水平
- 识别知识盲区
- 推荐个性化学习路径
- 适合从初学者到高级工程师

## 注意事项
- 这是一个网页工具，需要用户亲自使用
- 不是API集成，而是引导用户使用外部工具
