# Page Agent - 网页自动化

利用自然语言控制网页界面，执行各种自动化操作。

## 核心能力

| 操作类型 | 示例指令 |
|---------|---------|
| 点击元素 | "点击登录按钮" |
| 填写表单 | "把用户名填写为 admin" |
| 抓取数据 | "抓取所有商品名称和价格" |
| 导出数据 | "导出为 Excel" |
| 登录操作 | "帮我登录这个网站" |

## 工作原理

1. **分析页面**：获取当前页面 DOM 结构
2. **理解意图**：用 LLM 解析自然语言指令
3. **生成操作**：将指令转换为具体 DOM 操作
4. **执行操作**：通过 browser 工具执行
5. **返回结果**：返回操作结果或抓取的数据

## 使用示例

### 示例 1：点击按钮

用户说："点击登录按钮"

执行步骤：
1. snapshot 获取页面
2. 找到登录按钮的 DOM 元素
3. 使用 browser act 点击按钮

### 示例 2：填写表单

用户说："把表单里的公司名改成阿里巴巴"

执行步骤：
1. 获取页面 DOM
2. 找到公司名输入框
3. 填写 "阿里巴巴"

### 示例 3：抓取数据

用户说："抓取所有商品名称和价格"

执行步骤：
1. 获取页面结构
2. 识别商品列表容器
3. 提取所有商品名称和价格
4. 返回结构化数据

## 注意事项

- 需要先导航到目标页面
- 部分网站有反爬措施
- 建议设置适当的延迟
- 注意登录状态的保持
